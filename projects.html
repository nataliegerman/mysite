Matching Game "Svin the Cat"

Simple matching game to practice JavaScript as well as DOM handling and some CSS.

This game is using the picture of my cat - Svin. The left and right sides of the game's screen are identical, except for one thing: the left side has one extra cat. The user needs to click on that extra cat. If anything except the correct face is clicked, a message is displayed saying that the game is over. If the correct Svin's picture is clicked, all the currently displayed cats are deleted and a new set of cats is shown at random positions. Each time a new set of faces is shown there will be 5 more faces than before, on both the left and the right sides. There will always be one extra face shown on the left.
